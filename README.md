PokéGo
L'application permet de saoir à quel pokémon vous correspondez.
Pour cela vous devez disposer d'une connexion internet sur votre machine.
Il faut que vous installier le logiciel Go sur l'url: https://go.dev/doc/install.
Télécharger le projet sur: https://drive.google.com/drive/folders/1ae7TpndDTWQ4W2DEgQg6yR2tp2esdXMC.
Ensuite ouvrir un cmd et placer vous dans le répertoire de votre projet.
Taper la commande go run . puis rendez vous sur un navigateur et rentrer l'url: http://localhost.8080.
Voilà maintenant faites le test.
